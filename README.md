# Mane_Kelsa
Mane-Kelsa is a hyper-local Android app connecting domestic workers and gardeners with nearby residents. Workers can create profiles, update daily availability, and receive direct calls from employers. The platform improves job visibility, reduces unemployment gaps, and supports self-employment in small towns.
